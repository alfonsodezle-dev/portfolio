# 🌐 Portfolio Personal

Este proyecto es una **landing page de portafolio personal**, diseñada para presentar mis habilidades, proyectos y una forma directa de contactar conmigo. La página está construida con **HTML, CSS y JavaScript**, y ofrece una experiencia moderna, accesible y adaptable a cualquier dispositivo.

## ✨ Características principales

- **Presentación personal** con un párrafo que resume mis habilidades y enfoque profesional.
- **Interruptor de modo oscuro / modo claro**, implementado con JavaScript mediante un botón toggle.
- **Sección de proyectos** con enlaces directos a dos desarrollos destacados:
  - 📝 **Gestor de tareas** usando LocalStorage (HTML, CSS, JS)
  - 🌦️ **App del clima** consumiendo datos mediante Fetch API (HTML, CSS, JS)
- **Formulario de contacto** al final de la página, donde el usuario puede escribir su correo electrónico y un mensaje. Al enviar, recibo automáticamente la información en mi correo.

## 🚀 Proyectos destacados

### 📝 Gestor de Tareas (LocalStorage)
Aplicación sencilla y funcional que permite crear, editar y eliminar tareas, almacenándolas de forma persistente en el navegador mediante **LocalStorage**.

---

### 🌦️ App del Clima (Fetch API)
Aplicación que consulta una API meteorológica para mostrar el clima actual de cualquier ciudad. Utiliza **Fetch API** para obtener datos en tiempo real.

---

## 📩 Contacto

Al final de la página encontrarás un formulario donde puedes introducir tu **correo electrónico** y un **mensaje**.  
Cuando pulses el botón de enviar, recibiré automáticamente tu mensaje en mi bandeja de entrada.

---

## 🌓 Modo Oscuro / Claro

La página incluye un botón en la parte superior que permite alternar entre **modo oscuro** y **modo claro**, implementado con JavaScript mediante un toggle que modifica dinámicamente los estilos.

---

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- LocalStorage
- Fetch API
